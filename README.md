## Project Overview

**Mock-Mate** is a developer-focused web application that helps professionals prepare for technical interviews using AI-powered mock interviews. The platform uses Google's Gemini API to simulate realistic interview scenarios, with support for both conversational and rapid-fire question modes.

Whether you're preparing for your first job interview or brushing up your skills before a new role, Mock-Mate provides a personalized, interactive interview experience without the pressure of a real interview setting.

---

## Key Features

- **Google Authentication** – Secure login using Google accounts to save and track interview history
- **Chat Mode** – Natural conversational interview with an AI interviewer that adapts to your responses
- **Rapid-Fire Mode** – Quick-fire technical questions with immediate feedback for rapid skill assessment
- **Resume Context** – Upload your resume to get interview questions tailored to your specific experience and skills
- **Voice Support** – Use speech recognition to answer questions naturally, just like in real interviews
- **Interview History** – Save and review past interviews, track progress over time
- **Firestore Backend** – Secure cloud storage for all interview sessions and chat history
- **Responsive Design** – Fully responsive UI that works seamlessly on desktop and mobile devices

## Usage

### Chat Mode
1. Log in with your Google account
2. Navigate to Chat Mode
3. Click "New Chat" to start a conversation
4. Interact naturally with the AI interviewer
5. Use the microphone button for voice input (optional)
6. Review saved chats in the history sidebar

### Rapid-Fire Mode
1. Select the topic/role (e.g., "Frontend Engineer", "Data Structures")
2. Click "Start Game"
3. Answer 5 rapid-fire technical questions
4. Get instant feedback and a performance report
5. View detailed analysis of your answers

### Resume Mode
1. Upload or paste your resume
2. Start a "Resume Interview"
3. The AI interviewer will ask questions specific to your experience
4. Answer as you would in a real interview
